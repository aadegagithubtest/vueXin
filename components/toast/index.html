# toast全局提示

# 基础用法



在事件绑定中使用 $toast调用,可以通过设置 type 设置消息的(目前三种)的样式
默认是自动关闭的，可以通过 autoClosed设置弹框的自动关闭 
<x-button @click="showMsg1">默认消息</x-button>
<x-button @click="showMsg2" type="success">成功消息</x-button>
<x-button @click="showMsg3" type="warn">警告消息</x-button>
<x-button @click="showMsg4" type="error">错误信息</x-button>


export default {
    methods:{
     showMsg1() {
          this.$toast('默认的消息',{  
      })
                },
     showMsg2() {
         this.$toast('这是一条成功消息',{
          type:'success'
        })
       },
     showMsg3() {
         this.$toast('这是一条警告消息',{
          type:'warn'
         })
         },
     showMsg4() {
        this.$toast('这是一条错误消息',{
        type:'error'
       })
     },
  },
}



# 弹出的位置



全局消息弹框默认出现在顶部,可以通过设置 position 设置消息出现的位置(目前就三个) 
<x-button @click="showMsg1">默认顶部弹出</x-button>
<x-button @click="showMsg2" type="success">中间渐隐</x-button>
<x-button @click="showMsg3" type="warn">底部弹出</x-button>


export default {
     methods:{
       showMsg1() {
        this.$toast('顶部消息',{

        })
       },
       showMsg2() {
         this.$toast('中间消息',{
          type:'success',
          position:'center'
          })
        },
        showMsg3() {
         this.$toast('底部消息',{
          type:'warn',
          position:'bottom'
         })
       },
   },
}



# 关闭功能



默认是关闭按钮 wrong,可以通过 closeButton
设置【关闭】的文字或者Icon样式 
<x-button @click="showMsg1" type="success">默认</x-button>
<x-button @click="showMsg2" type="success">文字</x-button>
<x-button @click="showMsg3" type="success">其他Icon</x-button>


export default {
    methods:{
    showMsg1() {
      this.$toast('顶部消息',{
      type:'success',
     })
    },
    showMsg2() {
        this.$toast('文字',{
           type:'success',
           closeButton:{
           text:'关闭'
            }
        })
    },
    showMsg3() {
        this.$toast('dianzan Icon',{
         type:'success',
         closeButton:{
         closeIcon:'dianzan'
    }
      })
      },
   },
}



# HTML



HTMl插槽默认是关闭的,可以通过 enableHTML
设置为true开启。 
<x-button @click="showMsg1" type="success">默认</x-button>


export default {
     methods:{
     showMsg1() {
      this.$toast('<a href="https://www.google.com/">去谷歌</a>',{
      type:'success',
      enableHTML:true
      })
  },
   },
}


危险

你的站点上动态渲染的任意 HTML 可能会非常危险，因为它很容易导致 XSS 攻击。请只对可信内容使用 HTML 插值，绝不要对用户提供的内容使用插值。
